# Shomen Server

[Homepage](http://rubyworks.github.com/shomen-server) /

## Description

Real-time Documentation Server for all your precious gems.


## Instruction

Just run the command line utility.

    $ shomen-server

Then navigate to `http://0.0.0.0:3000`.
